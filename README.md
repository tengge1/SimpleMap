# SimpleMap
使用canvas和bing地图制作的简单地图，支持平移、缩放。

特点：

* 简单易懂，只支持平移和缩放。
* 使用es6最新语法，使用了类和继承，架构非常好，代码质量很高。
* 使用了canvas，数据量巨大时性能比svg要好。
